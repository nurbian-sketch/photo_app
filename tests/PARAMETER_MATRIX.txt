   │ Tv       │ ✅ Full  │ 🔒 Auto │ ✅ Full │
   │ P        │ ✅ Full  │ 🔒 Auto │ 🔒 Auto │
   └──────────┴──────────┴─────────┴─────────┘

2. EXPOSURE COMPENSATION:
   - Manual: LOCKED (you control everything directly)
   - Av/Tv/P: EDITABLE (adjust auto-exposure)

3. ALWAYS EDITABLE (regardless of mode):
   • ISO (28 choices)
   • White Balance (10 choices)
   • Color Temperature (76 choices)
   • Image Format (23 choices)
   • Picture Style (11 choices)
   • AF Method (6 choices)
   • Continuous AF (On/Off)
   • Drive Mode (6 choices)
   • Auto Lighting Optimizer (4 choices)

APPLICATION CONSTRAINTS (USER REQUIREMENTS):

1. ISO: Limited to max 800
   Available: [100, 200, 400, 800]

2. SHUTTER SPEED: Sensible ranges only
   Without Flash: 1/10 - 1/1000
   With Flash:    1/60 - 1/125

3. IMAGE FORMAT: Simplified to 3 choices
   [RAW, JPEG, RAW+JPEG]

4. PICTURE STYLE: 
   - Auto-set to "None" when RAW format selected
   - Show tooltip warning

5. DRIVE MODE:
   - Auto-configured to "Timer 2 sec" (required for remote)
   - Not shown in UI (background setting)
